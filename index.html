FAQ

Pertanyaan yang Sering Diajukan

Jawaban seputar fitur, paket, dan cara penggunaan Davcopia POS.

Apakah bisa digunakan tanpa internet?
Ya. Semua data tersimpan lokal terenkripsi. Transaksi, input barang, dan laporan berjalan tanpa internet. Internet hanya untuk login pertama (OTP), cloud backup, dan sinkronisasi paket.
Apa perbedaan paket Free, Basic, dan Pro?
Free — manajemen barang saja.
Basic — transaksi, laporan, hingga 3 kasir, struk digital.
Pro — semua fitur Basic + cloud backup, 10 kasir, import/export Excel, kustomisasi struk.
Apakah ada trial gratis?
Ya! Trial 7 hari gratis untuk fitur transaksi, laporan, dan kustomisasi struk. Satu kali per perangkat. Setelah habis kembali ke Free.
Berapa maksimal kasir?
Basic: 3 kasir aktif. Pro: 10 kasir. Setiap kasir punya PIN dan hak akses terpisah.
Bagaimana cara backup data?
Pengguna Pro: Pengaturan → Pencadangan → "Cadangkan". Restore dengan cara yang sama. Backup manual dan terenkripsi.
Apakah data aman?
Sangat aman. Enkripsi AES-256, login OTP WhatsApp, Device ID unik per perangkat. Cloud backup juga terenkripsi.
Bagaimana cara upgrade?
Klik tombol upgrade di halaman harga atau di aplikasi (Pengaturan → Status Aplikasi). Admin akan hubungi via WhatsApp untuk pembayaran dan aktivasi.
Bagaimana jika ganti HP?
Pro: backup → install di HP baru → login → restore. Basic: export data → pindah file → import kembali.
