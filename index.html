FAQ

Pertanyaan yang Sering Diajukan

Temukan jawaban instan seputar penggunaan, kendala teknis, dan lisensi fitur Davcopia POS.

Apakah Davcopia bisa digunakan tanpa internet?
Ya, Davcopia mengadopsi sistem offline-first penuh. Anda dapat melakukan input barang, transaksi kasir, dan melihat riwayat penjualan harian tanpa koneksi internet sama sekali.
Bagaimana cara backup data?
Untuk pengguna paket Basic, backup dapat dilakukan secara lokal berupa export data file. Untuk paket Pro, sistem akan otomatis mencadangkan data terenkripsi langsung ke cloud server kami secara real-time saat terhubung internet.
Apakah data bisnis aman?
Sangat aman. Data Anda disimpan di lokal penyimpanan enkripsi perangkat dan dicadangkan dengan protokol keamanan cloud terproteksi end-to-end tanpa risiko kebocoran data pihak ketiga.
Berapa maksimal kasir yang didukung?
Paket Basic mendukung hingga 2 kasir dalam single device, sedangkan untuk paket Pro tidak memiliki batasan (Unlimited) jumlah kasir serta mendukung integrasi multi-perangkat.
Bagaimana cara upgrade ke Pro?
Anda dapat langsung menekan tombol 'Upgrade ke Pro' pada aplikasi atau dashboard, lalu ikuti metode billing petunjuk pembayaran yang tersedia untuk aktivasi instan fitur pro.

Siap Mengembangkan Bisnis Anda?

Bergabunglah bersama ribuan wirausaha lainnya yang telah mendigitalisasi operasional kasir mereka agar lebih cepat, modern, dan efisien.
