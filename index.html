FAQ

Pertanyaan yang Sering Diajukan

Jawaban lengkap seputar fitur, paket, dan cara penggunaan Davcopia POS.

Apakah Davcopia bisa digunakan tanpa internet?
Ya, Davcopia menggunakan arsitektur offline-first. Semua data tersimpan di penyimpanan lokal perangkat yang terenkripsi. Anda bisa melakukan transaksi kasir, input barang, dan melihat laporan penjualan tanpa internet sama sekali. Internet hanya diperlukan saat login pertama kali (verifikasi OTP), cloud backup, dan sinkronisasi status paket.
Apa perbedaan paket Free, Basic, dan Pro?
Free — hanya bisa mengelola data barang (tanpa transaksi dan laporan).
Basic (Rp 50.000/bln) — transaksi kasir, laporan penjualan, hingga 3 kasir, hak akses, struk digital.
Pro (Rp 100.000/bln) — semua fitur Basic + cloud backup & restore, hingga 10 kasir, import/export Excel, kustomisasi struk, akses laporan untuk kasir, dan prioritas support.
Apakah ada trial gratis?
Ya! Setiap pengguna baru mendapat kesempatan Trial 7 hari gratis. Selama trial, Anda bisa menikmati fitur transaksi, laporan penjualan, dan kustomisasi struk. Trial hanya bisa digunakan satu kali per perangkat. Setelah trial berakhir, akun kembali ke mode Free.
Berapa maksimal kasir yang didukung?
Paket Basic mendukung hingga 3 kasir aktif, sedangkan paket Pro mendukung hingga 10 kasir. Setiap kasir memiliki PIN login sendiri dan hak akses yang bisa diatur oleh admin (akses ke menu barang dan laporan bisa diaktifkan/dinonaktifkan).
Bagaimana cara backup dan restore data?
Untuk pengguna Pro, buka menu Pengaturan → Pencadangan & Perangkat → tekan "Cadangkan" untuk upload data ke cloud server kami. Untuk restore, tekan "Pulihkan" dan data terakhir akan di-download dan diterapkan. Backup dilakukan secara manual untuk memberi Anda kontrol penuh kapan data dicadangkan. Semua data backup terenkripsi.
Apakah data saya aman?
Sangat aman. Semua file data lokal dienkripsi menggunakan algoritma AES-256 CBC dengan kunci unik per pemilik toko. Setiap perangkat terdaftar dengan Device ID unik dan akun dilindungi dengan OTP WhatsApp. Cloud backup juga terenkripsi dan hanya bisa diakses oleh pemilik akun.
Bagaimana cara upgrade paket?
Buka menu Pengaturan → Status Aplikasi → pilih paket yang diinginkan → kirim permintaan upgrade. Admin kami akan menghubungi Anda melalui WhatsApp untuk proses pembayaran dan aktivasi. Anda juga bisa menggunakan kode promo jika tersedia untuk mendapatkan diskon.
Bagaimana jika saya ganti HP?
Untuk pengguna Pro: lakukan backup ke cloud di HP lama, lalu install Davcopia di HP baru, login dengan nomor WhatsApp yang sama, dan pilih "Pulihkan" untuk mendownload semua data. Untuk pengguna Basic: Anda bisa export data sebagai file backup lokal, pindahkan file ke HP baru, lalu import kembali.
